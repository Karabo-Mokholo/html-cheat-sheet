HTML Summary

In this page you will get all the necessary tools to begin your web development journey, starting off with HTML. It is the building block of all modern web pages. Think of it as the structure of a house, or a skeleton of a human.

Who is HTML for?

HTML is for anyone who just wants to know the basics of websites. It includes :

  • Front-End Developers
  • Software Engineers
  • Freelancers
  • And many more...

Mastering HTML will help you not only with the basics, but it will be very useful should you wish to further your studies into stacks like MERN, or PERN

This page will give you all the basics of HTML and some useful tips as well.

HTML Elements

An element in html has the following structure :

<p>Hello World</p>
each syntax in this line has a meaning :

  • <p> - Opening Tag
  • Hello World - Content
  • </p> - Closing Tag

NB: Some elements are "Self-closing" meaning they do not have / do not require a closing tag. You can find a better detailed explanation here.

Exaples of these are : <img>, <input> and so on.
